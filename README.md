# Simple JavaScript Calculator

This is a working calculator I built using HTML, CSS, and JavaScript while learning how functions, user input, and DOM manipulation work together.

It is not just a UI project — it helped me understand how JavaScript controls logic behind real interactions like updating values, handling calculations, and resetting states.

---

## 🚀 What the Calculator Does

- Performs basic arithmetic operations (+, -, *, /)
- Updates the display in real time as buttons are clicked
- Calculates results using JavaScript logic
- Allows deleting last input (backspace)
- Clears all input when needed

---

## 🧠 What This Project Taught Me

- How to structure logic using functions instead of repeating code
- How the DOM updates based on user interaction
- How input builds up step by step before calculation
- How JavaScript evaluates expressions and returns results
- How state changes after pressing "=" affects future input

---

## ⚙️ Key Logic Concepts Used

- Function-based structure (`updateCalculation`, `calculateResult`, `displayCalculation`)
- String building for input handling
- Conditional logic for result handling
- Basic state management (tracking when calculation is finished)
